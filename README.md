# Covid Simulation
This project was created for studying purposes. It was chosen from Stanford Nifty Assignments http://nifty.stanford.edu/

### Summary
A graphical Java application that simulates the spread of COVID-19. Original assignment: http://nifty.stanford.edu/2021/bitner-covid-sim/

### Modifications
This project has an additional settings window that was not present in the original assignment. This new settings window lets the user to change various aspects of the simulation.
